Описание:


Необходимо реализовать слайдер с пагинацией по макету. 


1. Кнопки слайдера кликабельные и переключают между слайдерами
2. Кнопки слайдера имеют анимацию по макету в фигме
3. Слайдер должен иметь плавные анимации 
4. Слайдер должен быть адаптивен на промежуточных разрешениях между теми, что предоставлены в макете
5. У буллетов должна быть анимация по состоянию в макете

Задание должно быть реализовано со стеком:


* Next JS 15 (App Router)
* TailwindCSS
* Typescript


Слайдер должен пререндериться на сервере. При первичном запросе проект должен отдавать html с семантической версткой слайдера.


Любые дополнительные библиотеки не запрещены, но их чрезмерное количество не приветствуется

Код должен быть типизирован и структурирован на модули. Структура должна быть чистой, расширяемой, понятной. Документация приветствуется, но не обязательно


Тестовое задание должно отдавиться гитхаб-репозиторием с заполненным Readme по проекту. У репозитория должен быть подключен деплой на vercel
